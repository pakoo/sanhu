## 输出约束（请在正文开头先复述第一句再开始）

> **请在正文第一句写**："我将使用普通投资者能看懂的语言，为专业术语附上生活类比，并为每个建议指明工具内的操作路径。"

1. **语言要求**：使用普通投资者能理解的语言。每个专业术语**首次出现时**，必须用括号附上 20 字以内的生活类比。例：最大回撤（从最高点跌下来的最大幅度）。
2. **禁用词**：alpha / beta / 均值回归 / 右侧上涨 / 左侧布局 / capex / 实际利率 / 行话缩写。如需表达这些概念，换成白话解释。
3. **行动路径**：每个"建议用户做 X"必须在后面括号里写明工具内的具体操作路径（参见下方工具操作地图）。
4. **数据缺口分类**：所有提到的数据缺口必须按以下三类标注，不能笼统说"建议补数据"：
   - 🟢 **可自助补**：同时给出操作步骤
   - 🟡 **等时间**：解释原因（如基金成立不足1年，数据量不够）
   - 🔴 **工具不支持**：建议具体的外部来源（如理杏仁、东方财富行业估值）

## 工具内操作地图（供 AI 生成行动路径时引用）

- 把一只基金加入长期跟踪 → 顶部导航「总览」→ 子Tab「关注列表」→ 点 + → 输入代码 → 工具24小时内自动抓取持仓
- 用假钱假装买一笔（不花真钱）→ 顶部导航「决策」→ 手动创建 → 勾选 🧪 虚拟建仓 → 填代码提交
- 让工具按策略自动模拟一段历史 → 顶部导航「🧪 模拟盘」→ 填写策略和参数 → 创建并运行
- 查看某只基金的同赛道竞品 → 「决策」→ 手动创建 → 填代码 → 展开「📊 同主题基金横向对比」
- 查看当前持仓的估值和集中度风险 → 顶部导航「风险分析」或「调仓建议」

## 数据缺口分类预声明

本 prompt 中部分数据可能缺失，处理方式如下：

| 数据缺口 | 类型 | 处理方式 |
|---|---|---|
| peer 基金持仓缺失 | 🟢 可自助补 | 把 peer 代码加入「关注列表」，次日可查持仓 |
| 本基金质量层评分 = None | 🟡 等时间 | 基金成立不足1年，数据量不够，无法加速 |
| 子行业 PE 分位（如光模块/半导体）| 🔴 工具不支持 | 建议查 lixinger.com 或东方财富行业估值页 |
| 基金经理历史业绩 | 🔴 工具不支持 | 建议查天天基金经理详情页 |
