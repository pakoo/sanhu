🧪 模拟盘是什么？
选一个买入策略，假设从某天开始买，看如果真按这个策略买，历史上会赚多少（或亏多少）。买卖由策略自动执行，你只需要最后看结果。
📌 适合场景：听说"每月定投"效果不错 → 用模拟盘验证 2024 年实际结果 → 看数据再决定要不要用
示例： 策略选 dca_monthly → 基金填 009049 → 初始资金 15000 → 每月买 1000 → 时间 2024-01-01 到 2024-12-31 → 点"创建并运行" → 查看全年12笔交易和最终收益
⚠️ 模拟时长不足3个月时，结果没有统计意义，仅供参考和训练交易纪律。
