# 散户 sanhu

> 普通人买基金的决策助手。帮你看懂、帮你比较、帮你模拟 —— 最终由你决定。

---

## 这是什么

支付宝里买了几只基金，但不知道买得好不好？每次追涨杀跌，事后总是后悔？

sanhu 是一个**本地运行的基金投资辅助工具**，连接公开数据，帮普通投资者做三件事：

1. **看懂自己的持仓** — 当前盈亏、仓位分布、各基金综合评分
2. **做决策前先模拟** — 虚拟建仓观察，策略回测验证，不花真钱先试
3. **生成 AI 分析 Prompt** — 把你的真实持仓数据整理成结构化 Prompt，发给 Claude / ChatGPT 获取分析

工具不替你下单，不接入任何 AI 后端，不存储你的账户信息。

## 功能一览

| 模块 | 功能 |
|---|---|
| **持仓总览** | 总市值、盈亏、资产配置饼图、各基金7天/1月/3月走势 |
| **关注列表** | 自选基金跟踪，30天走势迷你图，综合评分徽章 |
| **决策记录** | 记录买入计划，支持🧪虚拟建仓（假钱观察真结果） |
| **策略模拟** | DCA定投 / 估值加权等策略，选时间段回测 |
| **风险分析** | 持仓行业集中度、最大回撤、波动率分析 |
| **调仓建议** | 对照目标配置，自动识别超配/低配 |
| **选基向导** | 问答式引导，从 8000+ 基金中筛选候选 |
| **AI 入口** | 生成数据丰富的持仓分析 / 赛道分析 Prompt |
| **市场估值** | 沪深300 / 中证500 PE百分位，历史分位实时显示 |

## 快速开始

**环境要求**：Python 3.9+，推荐使用系统自带 Python（`/usr/bin/python3`）

```bash
# 1. 克隆仓库
git clone https://github.com/pakoo/sanhu.git
cd sanhu

# 2. 安装依赖
/usr/bin/python3 -m pip install -r requirements.txt

# 3. 初始化数据库（自动加载内置市场数据，无需等待网络下载）
/usr/bin/python3 scripts/init_db.py

# 4. 启动服务
/usr/bin/python3 app.py
```

浏览器打开 [http://localhost:8000](http://localhost:8000)

## 录入持仓

有两种方式把你的基金数据导入工具：

**方式 A — 截图导入（推荐）**
1. 打开支付宝 → 基金账户 → 截图持仓页面
2. 点工具右上角「从截图导入」，上传截图
3. 工具自动识别基金代码、持有金额、盈亏

**方式 B — 手动录入**
1. 点「记录交易」→ 填入基金代码和买入金额
2. 逐笔录入历史交易

## 技术栈

- **后端**：Python + FastAPI + SQLite
- **前端**：原生 HTML/CSS/JS（无框架依赖）
- **数据源**：[akshare](https://github.com/akfamily/akshare)（公开基金数据）
- **图表**：ECharts

数据全部本地存储，不上传任何个人信息。

---

## 项目设计原则

- **工具辅助，用户执行**：工具帮你分析，买卖由你决定，不接入任何自动交易接口
- **可达性不降级专业深度**：术语旁有 ⓘ 说明，但原始数据不被简化替换
- **无后端 AI**：不调用任何 LLM API，AI 功能通过生成 Prompt 让用户自己去问

## 数据说明

内置的 `data/seed.sql` 包含：
- 28,000+ 基金基本信息（来自公开数据源）
- 近10年沪深300 / 中证500 PE历史
- 部分基金净值历史和持仓穿透数据

市场数据会在服务运行期间自动按调度更新（每日盘后）。
