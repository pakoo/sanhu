## 第一件事：导入你的持仓

打开工具后，你看到的是一个空白页面。需要先把支付宝里的基金数据导进来。

### 截图导入（推荐）

1. 打开支付宝 → 基金 → 持有页面
2. 截图（确保所有基金都在画面里，可以分多张截图）
3. 点页面右下角「从截图导入」→ 上传截图
4. 工具自动识别基金名称、持有金额、持有收益
5. 核对数据 → 点「确认导入」

**注意事项：**
- 支持支付宝「持有」列表视图的截图
- 如果某只基金识别有误，可以在核对表里手动修改
- 基金代码匹配是模糊匹配，确认导入前留意下拉框选的对不对

### 手动录入

点「记录交易」→ 搜索基金 → 填写买入日期、金额 → 保存。适合逐笔记录历史交易。

## 总览：看懂你的持仓

导入数据后，总览页会显示四项指标：

| 指标 | 含义 |
|---|---|
| **总资产** | 所有基金当前市值加总 |
| **持有收益** | 当前赚了或亏了多少钱，以及收益率 |
| **昨日收益** | 昨天一天赚/亏了多少 |
| **持有基金数** | 按债券/股票/混合/QDII 分类统计 |

### 资产配置饼图

展示你的钱分布在哪些类型的基金里。一般来说：
- 债券型占比大 → 偏保守
- 股票型占比大 → 偏激进
- 混合搭配 → 比较均衡

### 各基金收益率柱状图

一眼看出哪只基金表现好、哪只拖后腿。红色/绿色柱子分别代表正/负收益。

### 持仓明细表

每只基金一行，展示：
- **持有金额** / **持有收益** / **收益率**：盈亏情况
- **昨日收益**：昨天的涨跌
- **近7天 / 近1月 / 近3月**：不同时段的涨跌幅
- **近1月走势**：迷你折线图，直观看趋势
- **综合评分**：0-100 分，越高越好（详见下方说明）

> 看到不懂的术语，留意文字旁边的 **ⓘ** 图标，点击会弹出白话解释。

### 综合评分怎么看

综合评分是系统根据基金的业绩、风险、同类排名等多维度计算的打分：

- **70 分以上**（绿色）：近期表现较好
- **40-70 分**（灰色）：表现中等
- **40 分以下**（红色）：近期表现较弱

评分会随着市场变化而更新，不代表永久好坏。适合作为横向比较的参考。

## 关注列表：盯住感兴趣的基金

关注列表和持仓明细并列，在同一个区域的第二个 Tab。

**用途**：把想观察但还没买的基金加进来，持续跟踪它的表现。

**操作**：
- 点「+ 添加基金」→ 搜索基金名称或代码 → 加入关注
- 关注后可以看到和持仓一样的数据：走势、收益率、评分
- 想取消关注：点那行最右边的删除按钮

**和持仓的区别**：关注列表里的基金不计入你的总资产和盈亏统计，纯观察用。

### 市场估值信号

关注列表下方显示沪深300 和中证500 的估值状态：

- **低估**（绿色）：PE 百分位在历史 30% 以下，市场整体便宜
- **合理**（黄色）：PE 百分位在 30%-70%
- **高估**（红色）：PE 百分位在 70% 以上，市场整体偏贵

这个信号帮你判断大盘环境，不代表某只基金的涨跌。

## 风险分析：你的组合危险吗

**什么时候看这个页面**：打算追加投入之前，或者市场大跌时，检查持仓有没有风险集中的问题。

### 风险评分

系统综合计算的 0-100 分数：
- 分数越高 → 组合越稳健（偏债、分散）
- 分数越低 → 组合越激进（偏股、集中）

没有"正确"的分数，取决于你能承受多大波动。

### 相关系数热力图

展示你持有的基金之间「涨跌是否同步」：
- 红色格子 → 两只基金涨跌高度一致（分散效果差）
- 蓝色格子 → 两只基金涨跌相反（天然对冲）
- 白色格子 → 关系不大

如果整张图全是红的，说明你的基金虽然名字不同，但其实赚亏都是一起的。

### 风险预警

系统自动检测的问题提示，按严重程度分三级：
- 红色：需要关注的问题（如单一行业占比过高）
- 黄色：值得留意的情况
- 绿色：信息提示

## 持仓透视：你的钱到底买了啥

买了5只基金，但可能它们底层都在买同样的股票。这个页面帮你看穿基金的「包装」。

### 重叠持仓分析

列出被多只基金同时持有的股票。如果某只股票被你3只基金同时持有，说明你对这只股票的实际暴露比你以为的大得多。

- **出现只数**：这只股票被几只基金持有
- **实质暴露**：占你总持仓的加权百分比（超过 1% 会标红）

### 行业分布

把所有基金的底层持仓加权汇总，按行业显示。帮你回答「我的钱到底在哪些行业里」。

> 首次使用需要点「抓取持仓数据」，系统会从公开数据源获取各基金的持仓明细。

## 调仓建议：该加仓还是减仓

### 第一步：设定目标配置

用四个滑块设定你理想的资产比例（债券 / 股票 / 混合 / QDII），总和必须等于 100%。

没有标准答案，参考思路：
- 保守型：债券 70% + 混合 20% + 股票 10%
- 均衡型：债券 40% + 混合 30% + 股票 20% + QDII 10%
- 激进型：股票 50% + 混合 30% + QDII 20%

### 第二步：看对比图

系统自动对比当前配置和目标配置，生成建议：
- **超配**：这类基金买多了，考虑减仓
- **低配**：这类基金买少了，考虑加仓
- **适配**：比例合适，不用动

### 分月调整方案

如果偏差较大，系统会建议分几个月逐步调整，避免一次性大幅买卖。

## 选基：下一只买什么

这个页面帮你回答「我现在的持仓缺什么，应该补什么」。

### 第一步：自动诊断

系统分析你的当前持仓，找出配置缺口（比如「缺少新兴市场敞口」「港股配置不足」）。

### 第二步：回答几个问题

根据你的投资偏好回答选择题，比如风险偏好、投资期限等。

### 第三步：查看推荐

系统从 8000+ 只基金中筛选候选，每只显示：
- **综合分**：0-100 的评分
- **推荐理由**：为什么推荐这只
- **建议仓位**：建议投入金额占总资产的比例

看到感兴趣的，可以：
- 「加入关注」→ 先观察一段时间
- 「创建决策」→ 设定买入计划（可以用虚拟建仓先试）

## 策略回测：如果那时候开始定投

**用途**：想知道「如果我从2024年1月开始每月定投这只基金，现在赚了多少？」

### 操作步骤

1. 选择一只基金
2. 设定每期投入金额（如每月 1000 元）
3. 选择频率：每月 / 每两周 / 每周
4. 设定起止日期
5. 点「开始回测」

### 看结果

- **总投入 vs 最终价值**：投了多少钱，现在值多少
- **总收益率 / 年化收益率**：整体回报
- **最大回撤**：过程中从高点到低点最大亏了多少
- **收益曲线图**：每个月的资金变化走势

### 止盈定投回测

比普通定投多一个规则：收益率达到目标（如 15%）时自动止盈，然后重新开始定投。

可以对比「一直拿着」和「到点止盈」两种策略的差异。

## 模拟盘：不花钱先试试

和策略回测不同，模拟盘可以从今天开始「假装买入」，用真实的市场数据跟踪未来表现。

### 创建模拟

1. 选策略（目前支持：每月定投）
2. 选择基金
3. 设定初始资金和每月买入金额
4. 设定开始和结束日期
5. 点「创建并运行」

### 查看模拟结果

模拟创建后会出现在列表里，点开看详情：
- 每次买入的日期、金额、净值、份额
- 当前模拟组合的总收益率
- 和直接一次性买入的对比

**和策略回测的区别**：回测看历史，模拟盘看未来。回测告诉你「过去这样做会怎样」，模拟盘告诉你「从现在开始这样做会怎样」。

## 决策记录：记住你的每一次判断

每次想买或想卖的时候，先在这里记录下来，事后复盘。

### 创建决策

1. 选择基金
2. 设定计划金额
3. 设定止盈目标（涨多少卖）和止损目标（跌多少割）
4. 写下你的买入理由

### 虚拟建仓

勾选「虚拟建仓」后，系统会用虚拟资金模拟这笔交易。你不需要花真钱，就能看到如果真的买了会发生什么。

适合「想买但不确定」的情况 —— 先虚拟建仓观察 3 个月，确认判断对了再用真金白银执行。

### 告警追踪

设定了止盈/止损目标后，系统会自动追踪。当触发条件时，页面顶部会出现告警提示，提醒你做决定。

## AI 持仓解读：让 AI 帮你分析

点首页的「AI 持仓解读」按钮，工具会把你的持仓数据整理成一段结构化文本。

### 使用方法

1. 点按钮 → 系统生成包含持仓数据的分析 Prompt
2. 复制这段文本
3. 打开 [Claude](https://claude.ai) 或 ChatGPT → 粘贴 → 获取分析报告

### 为什么不直接内置 AI？

工具不调用任何 AI 接口。你的持仓数据只存在你的电脑上，不会发送给任何服务器。把数据交给 AI 的决定由你自己做，你能看到发送的每一个字。

## 常见问题

### 数据从哪来？

基金净值、行业分类等市场数据来自 [akshare](https://github.com/akfamily/akshare) 公开数据源。你的持仓数据完全保存在本地 SQLite 数据库中。

### 怎么更新数据？

点右上角「刷新数据」，系统会在后台更新所有基金的最新净值（约1-2分钟）。页面会自动刷新。

### 综合评分靠谱吗？

评分是量化参考，不是买卖信号。它综合了近期业绩、波动率、同类排名等因素，帮你快速筛选。但投资决策应该结合更多信息。

### 支持天天基金 / 蛋卷 的截图吗？

目前只支持支付宝持仓截图。其他平台的截图格式不同，识别可能不准确。

### 我的数据安全吗？

所有数据存储在你电脑上的 `data/` 目录里，工具不联网上传任何个人信息。你可以随时删除 `data/*.db` 文件清除所有数据。
